# Werwolf Spiel

Ein unterhaltsames Werwolf-Spiel in Python!

## Erste Schritte

Folge diesen einfachen Schritten, um das Projekt auf deinem Computer zum Laufen zu bringen.

### 1. Repository klonen

Öffne ein Terminal (CMD/PowerShell auf Windows, Terminal auf Mac/Linux) und gib folgende Befehle ein:

```bash
git clone https://github.com/hackepeter101/werwolf
cd werwolf
```

### 2. Virtuelle Umgebung erstellen

Eine virtuelle Umgebung ist wie ein separater Behälter für alle Abhängigkeiten dieses Projekts.

```bash
python -m venv venv
```

Dies erstellt einen neuen Ordner `venv` mit allem, was du brauchst.

### 3. Virtuelle Umgebung aktivieren

**Auf Mac/Linux:**
```bash
source venv/bin/activate
```

**Auf Windows:**
```bash
venv\Scripts\activate
```

Du wirst sehen, dass vor deiner Eingabeaufforderung `(venv)` erscheint. Das bedeutet, die Umgebung ist aktiv! ✓

### 4. Abhängigkeiten installieren (falls notwendig)

Wenn das Projekt eine `requirements.txt` Datei hat:
```bash
pip install -r requirements.txt
```

### 5. Das Spiel starten

```bash
python werwolf.py
```

Jetzt sollte das Spiel starten!

### 6. Virtuelle Umgebung deaktivieren

Wenn du fertig bist:
```bash
deactivate
```

---

## Tipps

- Du brauchst die virtuelle Umgebung nur **einmal** zu erstellen. Danach reicht es, sie jeweils zu aktivieren und zu deaktivieren.
- Die virtuelle Umgebung solltest du **nicht** in Git hochladen. Sie ist nur für deinen Computer.
- Wenn du mit dem Projekt fertig bist, gibst du einfach `deactivate` ein und die Umgebung ist wieder inaktiv.
